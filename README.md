# Sistema Modular de Tickets y Auditoría de Código

Aplicación web desarrollada con Vite + React + Tailwind y Firebase (Firestore),
orientada a la gestión de mantenimiento de software y auditoría estática de código.

## Características
- Linter básico para auditoría de código
- Gestión de tickets de mantenimiento
- Persistencia en Firebase Firestore
- Arquitectura modular
- Enfoque serverless (BaaS)

## Tecnologías
- React
- Vite
- Tailwind CSS
- Firebase Firestore
- Lucide Icons

## Ejecución
```bash
npm install
npm run dev
